1.) Install compiler:

sudo add-apt-repository ppa:ubuntu-toolchain-r/test

sudo apt update

sudo apt-get install gcc-9 g++-9

sudo update-alternatives --install /usr/bin/gcc gcc /usr/bin/gcc-9 900 --slave /usr/bin/g++ g++ /usr/bin/g++-9

(Special case: Raspberry Pi install

sudo update-alternatives --install /usr/bin/gcc gcc /usr/bin/gcc-8 800 --slave /usr/bin/g++ g++ /usr/bin/g++-8

sudo update-alternatives --install /usr/bin/gcc gcc /opt/gcc-9.1.0/bin/gcc-9.1 910 --slave /usr/bin/g++ g++ /usr/bin/g++-9 )

2.) Optional: Install CMake 3.14.7 ( see https://vitux.com/how-to-install-cmake-on-ubuntu-18-04/ )

sudo apt-get install libssl-dev

cd /opt

sudo wget https://github.com/Kitware/CMake/releases/download/v3.14.7/cmake-3.14.7.tar.gz

sudo tar -zxvf cmake-3.14.7.tar.gz

cd cmake-3.14.7

sudo ./bootstrap

sudo make

/usr/bin/cmake --version

next step only if [/usr/bin/cmake] exists and not a symlink

sudo mv /usr/bin/cmake /usr/bin/cmake-3.??.? [specify your version]

sudo make install

#sudo ln -s /opt/cmake-3.14.7/cmake/bin/cmake /usr/bin/cmake )

3.) install vcpkg (C++ package manager)

raspberry pi => https://github.com/microsoft/vcpkg/issues/9570

sudo apt-get install curl unzip tar

es empfehlen sich kurze Pfade , z.B. von /usr aus

git clone https://github.com/microsoft/vcpkg.git

cd vcpkg

optional:

git tag -l

git checkout 2019.11 (-f)

git fetch

optional: [only for qt5]

git remote add Neumann-A https://github.com/Neumann-A/vcpkg.git

git fetch Neumann-A

git merge Neumann-A/fix_qt_linux

sudo ./bootstrap-vcpkg.sh

./vcpkg integrate install

4.)optional: [only for qt5] Install libgtk2.0-dev and pkg-config:

sudo apt-get install libgtk2.0-dev pkg-config

sudo apt-get install libxi-dev libgl1-mesa-dev libglu1-mesa-dev mesa-common-dev libxrandr-dev libxxf86vm-dev

5.) Install boost , magic-enum und opencv with vcpkg

./vcpkg install magic-enum

sudo ./vcpkg install boost

sudo ./vcpkg install opencv

optional:[only for qt5]

sudo apt-get install libgl1-mesa-dev libglu1-mesa-dev

sudo ./vcpkg install qt5

6.) Install source code

git clone https://github.com/zweistein-frm2/charm.git

git clone https://github.com/palacaze/sigslot.git

git clone https://github.com/timniederhausen/asio-extensions.git

You should have the following directory structure:

|-----\asio-extensions

|

|-----\charm

|

|-----\sigslot

7.) Jetzt braucht man die Pfade aus 3.) die dann in die command line richtig eingefügt werden (also bei euch /home/alanghof oder users/alanghof mit dem bei euch vorhanden Pfad ersetzen)

mkdir hiersolleshin

cd Hiersolleshin

cmake -DCMAKE_TOOLCHAIN_FILE=/vcpkg/scripts/buildsystems/vcpkg.cmake -DASIO_EXTENSIONS_PATH=/mnt/c/users/alanghof/source/repos/asio-extensions/include -S /mnt/c/users/alanghof/source/repos/charm/ -B /mnt/c/users/alanghof/hiersolleshin/

make

in [fullpath to hiersolleshin] finden sich dann die Executables
